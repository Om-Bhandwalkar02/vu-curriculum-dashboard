# 🎓 Curriculum Analytics Dashboard

A comprehensive academic analytics platform developed for the **Computer Science Department** to evaluate, benchmark, and visualize curriculum evolution, flexibility, compliance, employability, industry alignment, and academic quality indicators across multiple curriculum revisions.

---

## 📌 Project Overview

The Curriculum Analytics Dashboard provides a centralized platform for analyzing curriculum revisions and measuring their impact through data-driven insights and interactive visualizations.

The system helps academic stakeholders evaluate curriculum effectiveness, identify gaps, monitor compliance, and support continuous curriculum improvement.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Om-Bhandwalkar02/vu-curriculum-dashboard.git
cd vu-curriculum-dashboard
```

### Create Virtual Environment

```bash
python -m venv .venv
```

### Activate Environment

Windows:

```bash
.venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Apply Migrations

```bash
python manage.py migrate
```

### Run Server

```bash
python manage.py runserver
```
